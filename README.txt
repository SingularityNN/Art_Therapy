=================================================================================================
Установка:
1. Скачать python с официального сайта (python.org)
2. Во время установки python поставить галочку напротив "добавить в PATH" (или что-то вроде того)
3. Запустить установщик
    для Windows - installer_windows.bat,
    для macOS/Linux - installer_mac_linux.sh
=================================================================================================



-------------------------------------------------------------------------------------------------
Запуск:
1. Запустить лаунчер
    для Windows - launcher_windows.bat,
    для macOS/Linux - launcher_mac_linux.sh

В браузере должна открыться страница с интерфейсом
